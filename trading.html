    About This Project

    .Net

    C#

    WPF

    This project is an automated trading application made with C# and WPF. It connects to DAS Trader using TCP and sends/recieves data by parsing space-delimited messages. The application will automatically log into the Trade Ideas account specified in the settings, and will scrape the page for stock symbols that appear on the scanner.

    All trading is done automatically and many of the parameters can be customzsed in the settings window. Customizable parameters include stop/start times, buy/sell triggers, loss limit, sell half/whole, as well as specific account information for DAS Trader and Trade Ideas.

    Features

    Automatically loads symbols from Trade Ideas stock scanner.

    Receives live market data and sends order request to DAS Trader by TCP connection.

    All settings are completly customizable in the settings menu.

    Trading is hands-free and fully-automated.

    Detects if power is switched to UPS and will exit all trades until main power is restored.
