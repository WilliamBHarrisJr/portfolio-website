    About This Project

    Java

    SQL

    JavaFX

    This application was built using Java, JavaFX, and SQL and schedules appointments between employees and clients. All data is stored and retrieved from a database and can be accessed from anywhere in the world, with times adjusted for the machine's local time zone. All customers and appointments can be added, deleted, and changed within the application, with changes reflected in the database. Customized reports can be generated and viewed in the application. Users must log in to the application to ensure data integrity and security.

    Explore It's

    Features

    Automatically adjusts appointment times according to the user's time zone.

    Prevents over-lapping appointments from being scheduled.

    All customers can be added, deleted, and changed in the database.

    Customized reports can be generated and viewed.

    Login is required in order to protect data integrity and confidentiality.
