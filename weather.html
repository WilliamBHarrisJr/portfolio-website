    About This Project

    Python

    Custom Tkinter

    This application was built using Python and Custom Tkinter and uses machine learning to make weather predictions. It takes historical weather data in the form of a CSV file and uses the Pandas library to create a logistical regression model from the data. The application then takes weather parameters as user input and uses the data model to make a prediction. To help visualize the data, the user can view a histogram, scatter-plot matrix, and a pie chart.
